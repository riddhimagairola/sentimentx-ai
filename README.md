# SentimentX AI

An AI-powered web application to analyze homestay customer reviews.

## Features
- Sentiment Analysis (Positive/Neutral/Negative)
- Theme Detection
- AI Response Suggestions

---

## Tech Stack
- React JS
- Node.js (Express)
- MongoDB
- AI APIs (Gemini/OpenAI)
